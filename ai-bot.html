Personal AI Bot Using Botpress

✕

This project is a personalized AI bot trained on my resume and portfolio details. Built using Botpress and OpenAI, it allows visitors to ask questions like:

• “What are Karthik’s skills?”
• “Tell me about his projects in IoT.”
• “Where did he study?”

The bot is hosted online and trained with FAQs, resume content, and personalized replies using knowledge base integration. It can be used by recruiters, visitors, or collaborators to quickly understand my background and capabilities.

Features:

  • Botpress-based no-code visual flow setup
  • Custom knowledge base from PDF/resume
  • OpenAI GPT integration
  • Deployed and embeddable on any page

Future improvements include analytics on bot usage, voice assistant capability, and multilingual responses.
